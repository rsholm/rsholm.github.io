# Daily Calendar

Daily Calendar is an Android app that turns an old phone or tablet into a simple daily calendar dashboard. It displays a user's schedule for the current day using read-only calendar access.

The app is designed for kiosk-style use on a desk, nightstand, wall mount, or tablet stand. It can show a full-day overview or a zoomed view, supports light and dark mode, and remembers the user's display preferences.

## App Information

| Item | Details |
|---|---|
| App name | Daily Calendar |
| Android package name | `com.dailycalendar` |
| Platform | Android |
| Primary purpose | Display the user's daily calendar events in a dashboard view |
| Google API used | Google Calendar API |
| Google OAuth scope requested | `https://www.googleapis.com/auth/calendar.readonly` |

## Features

- Shows today's calendar events in a clean day-view layout
- Supports Full Day and Zoom views
- Supports Light and Dark mode
- Remembers the user's last selected view and theme
- Designed for tablets and phones used as always-on calendar displays
- Uses read-only calendar access only

## Why Daily Calendar Requests Google Calendar Access

Daily Calendar requests read-only Google Calendar access so the app can display the user's own calendar events in the daily dashboard.

Daily Calendar only reads calendar information needed to show the schedule, such as event titles and event times. The app does not create, edit, delete, sell, share, or transfer calendar events.

Daily Calendar requests the following Google OAuth scope:

```text
https://www.googleapis.com/auth/calendar.readonly
```

This is the minimum Google Calendar scope needed for the app's core functionality.

## Google API Limited Use Disclosure

Daily Calendar's use and transfer of information received from Google APIs adheres to the [Google API Services User Data Policy](https://developers.google.com/terms/api-services-user-data-policy), including the Limited Use requirements.

Daily Calendar does not use Google user data for advertising. Daily Calendar does not sell Google user data. Daily Calendar does not use Google user data to train artificial intelligence or machine learning models.

This Privacy Policy explains what information the App accesses, how it is used, and what choices users have.

## 1. Information the App Accesses

Daily Calendar may request access to the following information.

### Google Calendar Data

If a user connects their Google account, the App requests read-only access to Google Calendar using this Google OAuth scope:

```text
https://www.googleapis.com/auth/calendar.readonly
```

This allows the App to read calendar event information such as:

- Event titles
- Event start and end times
- All-day event status
- Calendar event timing needed to display the daily schedule

The App does **not** create, edit, delete, or modify calendar events.

### Device Calendar Data

On Android devices, the App may request calendar permission in order to read calendar events available through the device's calendar provider.

### App Settings

The App may store basic local preferences on the device, such as:

- Dark mode or light mode selection
- Zoom or full-day view selection

These settings are stored locally on the user's device.

## 2. How Information Is Used

Calendar information is used only to display the user's daily calendar inside the App.

The App uses calendar data to:

- Show the user's schedule for the current day
- Display event times and event titles
- Refresh the daily schedule periodically

## 3. Data Sharing

Daily Calendar does **not** sell, rent, or share user calendar data with advertisers, data brokers, or third parties.

Calendar data is used only inside the App for the purpose of displaying the user's schedule.

## 4. Data Storage

Daily Calendar does not operate a backend server for storing users' calendar data.

Calendar information is retrieved for display in the App. Basic app preferences, such as dark mode and zoom mode, may be stored locally on the user's device.

## 6. Permissions

Users may revoke permissions through Android system settings or their Google Account permissions page.

## 7. User Choices and Controls

Users can:

- Decline calendar access, although the App may not be able to display calendar events
- Revoke calendar permission in Android settings
- Revoke Google account access from their Google Account permissions page
